# Doctor Appointment System

A responsive medical appointment booking website built using HTML, CSS, Bootstrap, PHP and MySQL.

## Features
- Patient Appointment Booking
- Admin Login Panel
- Database Integration
- Form Validation
- Responsive Design

## Technologies Used
- HTML
- CSS
- Bootstrap
- PHP
- MySQL

## How to Run
1. Install XAMPP
2. Import the database file into phpMyAdmin
3. Place project folder inside htdocs
4. Start Apache & MySQL
5. Open localhost/project-folder-name
